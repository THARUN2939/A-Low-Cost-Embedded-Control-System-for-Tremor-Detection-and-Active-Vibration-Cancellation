# Performance Evaluation

## Test Conditions

- Tremor Frequency: 3–12 Hz
- Controller: PD
- Sensor: MPU6050
- Actuator: Servo Motor

## Results

| Parameter | Result |
|-----------|--------|
| Tremor Reduction | 65–75% |
| Latency | <15 ms |
| Response | Stable |
| Power | Low |

The prototype successfully differentiated voluntary movement from tremors and provided smooth compensation.
