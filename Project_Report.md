# Project Summary

## Title

A Low-Cost Embedded Control System for Tremor Detection and Active Vibration Cancellation

## Objective

Develop an affordable embedded control system capable of detecting physiological tremors and suppressing them in real time.

## Technologies Used

- ESP32
- MPU6050
- Servo Motor
- Embedded C++
- Arduino IDE

## Outcome

The developed system successfully demonstrated active tremor cancellation while maintaining voluntary movement.
